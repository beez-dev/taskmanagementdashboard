# Presentation / Store

> **Note:** The Redux store has been moved to [`src/application/store/`](../../application/store/README.md).
>
> Presentation is a dumb-UI-only layer. State management (store, hooks, provider) is orchestration and belongs in the application layer.

This folder is intentionally empty. Any Redux slices that are purely UI state (sidebar open/closed, active filter) will live at `src/application/store/slices/` if and when they're added.
